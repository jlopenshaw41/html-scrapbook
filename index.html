Metadata tags

<html></html> - tells the browser to expect html

<head></head> - this section provides the browser with important info. It won't display on the page itself

<title></title> - the text you put inside this tag will appear on the browser tab

Block elements

<header></header> - contains all the elements of the header section of the webpage (semantic html)

<nav></nav> - contains the navigation menu (semantic html)

<main></main> - contains the main content of the webpage (semantic html)

<footer></footer> - contains the footer of the webpage

<section></section> - used to separate a section

<article></article> - used to separate an article

<aside></aside> - used to separate content that is not directly related e.g. bibliography, additional info, pull quotes (semantic html)

<h1></h1> - h1 to h6 tags are used to style/denote headings. h1 is the largest, h6 is the smallest

<p></p> - a paragraph

<div></div> - a container, used to group together related elements on a webpage

<input type="text"> - a text box for users to enter info

<input type="checkbox" name="" id=""> - a checkbox
